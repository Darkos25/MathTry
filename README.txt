Программа выполнена для умножения, вычитания и деления без использования операторов, кроме сложения
Первая часть кода производит умножение.
Далее в зависимости от большего числа введенного пользователем програма выполняет вычитание и деление в соответствующем порядке. 
Вычитание и деление ОБЯЗАТЕЛЬНО выполняется из большего меньшее.
