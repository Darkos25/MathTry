Программа выводит результат умножения, вычитания и деления двух чисел. Реализована без использования операторов (-,*,/)
Первая часть кода производит умножение.
Далее в зависимости от большего числа, введенного пользователем, программа выполняет вычитание и деление в соответствующем порядке. 
Вычитание и деление ОБЯЗАТЕЛЬНО выполняется из большего меньшее.
